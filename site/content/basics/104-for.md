## 知识点

Go 中 `for` 是唯一的循环关键字，支持多种形式。最基本的死循环写作 `for { ... }`，配合 `break` 使用。带条件的循环写作 `for condition { ... }`，类似其他语言的 while。标准的三段式循环写作 `for init; condition; post { ... }`，其中 `continue` 可跳过当前迭代。

Go 还支持使用标签（label）来控制循环的跳出。通过在循环前定义标签（如 `loop:`），可以在多层嵌套循环中使用 `break loop` 跳出指定的外层循环，而不是只跳出最内层循环。
