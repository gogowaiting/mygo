## 知识点

Go 的 `if` 语句不需要用括号包围条件表达式。`if` 支持带初始化语句的写法，变量的作用域仅限于 if/else 块内，这是 Go 中常见的惯用法。

Go 社区推荐的错误检查模式是先执行操作，再用 `if err != nil` 判断错误。结合初始化语句，可以写出紧凑的错误检查代码。此外，Go 中检查 map 的 key 是否存在通常也使用 `if v, ok := m["key"]; ok` 的形式。
