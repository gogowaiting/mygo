## 知识点

Go 的 map 是哈希表实现的键值对集合。使用 `make` 创建的 map 已分配内存空间，可以直接赋值。注意对 nil 的 map 赋值会触发 panic。map 支持字面量初始化，如 `map[string]string{"k1": "one"}`。

map 的常用操作包括：通过键读取值、使用 `delete` 删除键值对、使用 `len` 获取元素数量。判断键是否存在可以使用双返回值形式 `v, ok := m["key"]`，当 ok 为 false 时表示键不存在。此外，map 底层的切片扩容过程也值得观察，可以看到 len 和 cap 的增长规律。
