## Git 合并开源分支到私有仓库

1. 拉取开源仓库代码到本地
   ```bash
   git pull opensource_addr
   ```

2. 建立私有仓库为远程仓库
   ```bash
   git remote add upstream private_addr
   ```

3. 查看当前远程仓库
   ```bash
   git remote show
   ```

4. 拉取远程仓库 index
   ```bash
   git fetch upstream
   ```

5. 切换到需要合入私有仓库的开源项目分支上
   ```bash
   git checkout xxx
   ```

6. 将当前分支推送到私有仓库
   ```bash
   git push upstream HEAD:[tag]
   ```

如果还需要合入私有仓库分支上的私有 patch 到新合入的开源分支，在第 5 步之后执行，然后再执行第 6 步：

```bash
git cherry-pick xxx
```
